POST /api/code/new :

  Post a new code snippet.
  Example: 

        Request:
        
            curl --url https://www.cps.com/api/code/new
                --request POST
                 --header "Content-Type: application/json"
                 --data '{"username":"xyz","password":"xyz"}'
            
        
        Response:
        
            {
                "id": "Test",
                "date": "Test test"
            }
